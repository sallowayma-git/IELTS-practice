# Template for obfs4 bridges.
#
# Do not commit real bridges here. Publicly publishing bridges makes them less
# useful and leaks deployment details. Put private bridge lines in
# backend/tor/bridges.local.txt and set:
#
#   TOR_BRIDGES_LOCAL_FILE=./tor/bridges.local.txt
#
# Valid obfs4 lines are normalized and deduplicated by docker-entrypoint.sh.
# This template file alone does not enable bridge mode.
# Example:
# obfs4 127.0.0.1:443 FINGERPRINT cert=BASE64 iat-mode=0
