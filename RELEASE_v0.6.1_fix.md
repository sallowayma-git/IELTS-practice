# Release v0.6.1 fix

## 发布概述

本次版本是针对 `opensource` 分支核心特性的稳定发布版。在合并至 `release` 的过程中，我们不仅解决了复杂的合并冲突，还对生产环境进行了深度“瘦身”——移除了所有开发工具（developer）、模板（templates）及内部指令文档（AGENTS.md），确保提供一个纯净、高效的运行环境。

---

## 核心更新

### 🎨 视觉与交互 (HeroUI & Dynamic Background)
- **HeroUI 桥接集成**：引入了全新的 HeroUI 设计系统桥接层，整体提升了 UI 的现代感。
- **Three.js 动态背景**：实现了基于 WebGL 的动态背景系统，支持多种主题随动。
- **UI 细节精修**：优化了玻璃拟态（Glassmorphism）效果、滑块拖拽感以及 Emoji 图标的处理。
- **引导系统 (Onboarding)**：新增 Onboarding Tour 引导组件，帮助用户快速上手新功能。

### ⚡ 练习系统增强
- **计时逻辑统一**：彻底统一了 UI 显示与后台报告中的计时逻辑，解决误差问题。
- **套题选择门户**：重构了套题选择界面（Suite Selection Portal），支持更复杂的 UI 状态切换。
- **阅读模式优化**：修复了阅读考试内容的渲染问题及计时器的交互 Bug。

---

## Bug 修复
- 修复阅读答案键与计时器在极端情况下的失步问题。
- 修正主按钮样式在部分分辨率下的显示偏移。
- 解决成就图标在 Emoji 处理列表中的识别冲突。
