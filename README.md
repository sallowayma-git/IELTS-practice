# IELTS 练习系统使用说明

## 系统简介

这是一个本地化的IELTS（雅思）练习系统，包含P1、P2、P3三个级别的阅读练习题目。系统支持HTML和PDF两种格式的题目，具备完整的练习记录、数据管理和性能优化功能。

## 快速开始

### 1. 系统要求
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 支持JavaScript和本地存储
- 建议屏幕分辨率：1024x768或更高

### 2. 启动系统
1. 下载并解压系统文件到本地目录
2. 双击打开 `improved-working-system.html` 文件
3. 系统将自动加载题库并显示主界面

### 3. 基本操作
- **题库浏览**：点击"📚 题库浏览"查看所有题目
- **分类练习**：选择P1、P2或P3进行分类练习
- **练习记录**：查看历史练习记录和统计信息
- **系统设置**：进行系统配置和数据管理

## 主要功能

### 📚 题库管理
- **题目分类**：按P1、P2、P3分类显示
- **搜索功能**：支持按标题和分类搜索题目
- **文件格式**：支持HTML交互式练习和PDF查看
- **题目状态**：显示题目可用性和格式信息

### 📊 练习记录
- **自动记录**：练习过程自动保存到本地
- **统计分析**：显示练习次数、平均分数等统计信息
- **记录管理**：支持查看、编辑和删除练习记录
- **数据导出**：可导出练习数据进行备份

### 🔧 系统管理
- **数据备份**：自动和手动备份功能
- **性能监控**：实时监控系统性能和缓存状态
- **错误处理**：智能错误检测和恢复机制
- **系统检测**：全面的系统功能检测工具

## 详细使用指南

### 开始练习
1. 在主界面选择练习方式：
   - 点击"📚 题库浏览"查看所有题目
   - 点击"P1"、"P2"或"P3"进行分类练习
2. 在题目列表中选择要练习的题目
3. 点击"开始练习"按钮开始答题
4. 完成答题后点击"Submit"提交答案
5. 系统将自动记录练习结果

### 查看练习记录
1. 点击"📊 练习记录"进入记录页面
2. 查看练习统计信息和历史记录
3. 可以按日期、分数等条件筛选记录
4. 支持删除单个或批量删除记录

### 系统设置和维护
1. 点击"⚙️ 设置"进入设置页面
2. 可以进行以下操作：
   - **系统检测**：检查系统功能是否正常
   - **性能报告**：查看系统性能统计
   - **数据备份**：创建和管理数据备份
   - **数据导入导出**：备份和恢复数据
   - **缓存清理**：清理系统缓存

## 故障排除

### 常见问题

#### 1. 题目无法打开
**症状**：点击"开始练习"后页面无法加载或显示错误

**解决方案**：
- 检查浏览器是否允许弹窗
- 确认文件路径是否正确
- 运行系统检测工具进行诊断
- 查看浏览器控制台是否有错误信息

#### 2. 练习数据丢失
**症状**：之前的练习记录消失

**解决方案**：
- 检查浏览器存储设置
- 尝试从备份恢复数据
- 确认没有清理浏览器数据
- 使用数据完整性检查工具

#### 3. 系统运行缓慢
**症状**：页面加载慢，操作响应延迟

**解决方案**：
- 清理浏览器缓存
- 运行性能优化工具
- 检查系统资源使用情况
- 重启浏览器

#### 4. 移动设备显示异常
**症状**：在手机或平板上显示不正常

**解决方案**：
- 确认使用现代移动浏览器
- 检查屏幕方向设置
- 尝试刷新页面
- 使用横屏模式获得更好体验

### 高级故障排除

#### 系统检测工具
1. 进入"⚙️ 设置"页面
2. 点击"🔧 系统检测"
3. 等待检测完成并查看报告
4. 根据报告建议进行修复

#### 数据完整性检查
1. 在设置页面点击"🔍 数据检查"
2. 系统将验证所有数据的完整性
3. 如发现问题会提供修复建议
4. 可以尝试自动修复或手动处理

#### 重置系统
如果遇到严重问题，可以重置系统：
1. 备份重要数据（导出练习记录）
2. 点击"🗑️ 清除缓存"
3. 刷新页面重新开始
4. 如需要可导入之前的备份数据

## 技术说明

### 系统架构
- **前端技术**：HTML5、CSS3、JavaScript ES6+
- **数据存储**：浏览器本地存储（localStorage）
- **文件格式**：HTML练习页面、PDF文档
- **通信机制**：跨窗口消息传递

### 数据格式
- **练习记录**：JSON格式存储在localStorage
- **系统设置**：键值对形式存储
- **备份文件**：JSON格式，包含校验和

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+
