# Release v0.6.1

## 发布概述

本次版本带来61个提交，主要包含阅读考试系统重构、练习记录功能增强、存储架构升级等重要更新。

---

## 新增功能

### 阅读考试系统
- 新增统一阅读考试支持，集成多种题型
- 添加P3中等难度阅读考试"音乐语言"文章
- 支持表格填空题型样式
- 阅读考试数据支持多答案选项
- 更新Fishbourne Roman Palace阅读解析数据

### 练习记录系统
- 重构练习记录持久化存储架构
- 添加练习记录模态框显示功能
- 增加练习记录回放功能和视觉优化
- 支持练习记录删除链路和app.state同步

### 存储架构
- 实现持久化存储与偏好设置存储分离
- 添加会话初始化防重复机制
- 修复题库全量/增量重载路径硬编码问题

### UI/UX改进
- 集成文楷(WenKai)字体并应用温暖的纸张主题
- 添加引导流程功能并优化按钮样式
- 改进题库排序下拉菜单和套题模式选择界面样式
- 实现成就系统带等级结构和模态框UI
- 添加GPL许可弹窗处理

### 代码架构
- 从内联脚本提取核心应用到main.js
- 引入模块化服务和数据文件
- 统一高亮笔记模型
- 继续代码精简并添加练习页面回归测试
